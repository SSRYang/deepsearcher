# DeepSearcher - AI-Powered Web Research Assistant

DeepSearcher is a fully local web research and summarization assistant powered by HelloAgents. It combines the power of AI agents with web search capabilities to provide comprehensive research and analysis.

## Features

- **AI-Powered Research**: Leverages advanced AI agents for intelligent information gathering
- **Web Search Integration**: Integrates with multiple search providers including Tavily and DuckDuckGo
- **Local Processing**: All processing happens locally, ensuring privacy and security
- **Comprehensive Summarization**: Generates detailed reports and summaries from research findings
- **Modular Architecture**: Clean separation between backend services and frontend interface
