# 任务 3: 轻量部署实践

任务概览：探究2025年多模态大模型在边缘/端侧场景下的轻量化适配技术，聚焦量化-aware适配、稀疏化微调、硬件协同编译三大方向，重点关注ONNX/TensorRT生态及代表性开源项目。

来源概览：
- MachineLearningMastery《2025年大模型掌握路线图》：强调GGUF（llama.cpp）、ONNX与TensorRT为端侧推理核心工具链；
- NVIDIA官方文档（TensorRT Edge-LLM）：定义其为面向Jetson/DRIVE平台的C++高性能VLM/LLM推理运行时，支持FP8/INT4/NVFP4量化、LoRA、EAGLE、CUDA Graphs；
- GitHub仓库（NVIDIA/TensorRT-Edge-LLM）：提供Python导出流水线（HF→ONNX+量化）、C++引擎构建器、纯C++无Python依赖运行时；
- NVIDIA开发者论坛：确认TensorRT Model Optimizer与nvidia-modelopt为官方量化主力工具，支持PTQ与QAT，已实测FP4精度收敛；
- KodekX 2025指南：指出端侧部署需兼顾压缩模型、专用硬件加速与实时隐私保护三重目标。
